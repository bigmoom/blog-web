Vector 和 ArrayList 区别

相同点：

  1. 继承了相同的父类实现了相同的接口
  2. 底层都是数组实现的
  3. 初始默认长度都是10

不同点：

  1. Vector中public方法多数加了synchronized，线程安全，ArrayList没有
  2. 扩容方式不同，ArrayList扩展后数组大小为原数组的1.5倍，Vector中多了扩展因子，当扩展因子大于0的时候，扩展后为原数组大小加上扩展因子大小，否则为原数组的两倍。
