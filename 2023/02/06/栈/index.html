栈

栈的基础知识

栈是一种非常常见的数据结构，他的核心特点就是LIFO，后进先出。

对于栈，我们主要需要知道的是栈的三个主要函数（下面的栈我们使用ArrayList实现，所以对于容量为自动调整）

  • PUSH：即入栈，我们在栈顶放入元素，栈顶标识符++
  • POP：即出栈，我们取出栈顶元素，栈顶标识符–
  • PEEK：获取栈顶元素但不出栈，栈顶标识符不变

栈的应用

栈的应用其实主要就是后入先出的核心思想，当对于数据的存储顺序与使用顺序相反的时候即最后保存的数据最先使用，我们可以将这些数据存储到栈中

后缀（逆波兰）表达式

后缀表达式是一种算术表达式，他的操作符在操作数的后面，我们平时用的是中序表达式，即操作符在操作数中间。

例如下面式子：

4.99+5.99+6.99∗1.064.99+5.99+6.99*1.06 4.99+5.99+6.99∗1.06

我们自己肯定知道乘法的等级要高于加法，所以先算6.99∗1.066.99*1.066.99∗1.06再加上前面的和，但是对于计算器或者计算机，没有括号的辅助，很有可能就是直接按照顺序进行计算，就会造成错误。但是我们来看后缀表达式

4.99 5.99 + 6.99 1.06 ∗ +4.99\space5.99\space+\space6.99\space1.06\space*\space+ 4.99 5.99 + 6.99 1.06 ∗ +

上面的表达式表示我们先4.99+5.994.99+5.994.99+5.99计为A1A_1A1​，然后6.99∗1.66.99*1.66.99∗1.6计为A2A_2A2​，最后再将A1,A2A_1,A_2A1​,A2​相加

使用后缀表达式，就已经表明了计算的顺序，无须借用括号来表明。

计算后缀表达式的方法最容易的就是使用栈来操作：当遇到一个数，那么我们就把他入栈，遇到表达式弹出栈顶两个数进行计算，将结果入栈。

计算后缀表达式的值

LeetCode 150

给你一个字符串数组 tokens ，表示一个根据 逆波兰表示法 表示的算术表达式。

请你计算该表达式。返回一个表示表达式值的整数。

输入：tokens = ["2","1","+","3","*"]

输出：9

解释：该算式转化为常见的中缀算术表达式为：((2+1)∗3)=9((2 + 1) * 3) = 9((2+1)∗3)=9

中缀转后缀表达式

输入：tokens = ["a","+","b","*","c","+","(","d","*","e","+","f",")","*","g"]

输出：abc*+de*f+g*+

题目中中序表达式为

a+b∗c+(d∗e+f)∗ga+b*c+(d*e+f)*g a+b∗c+(d∗e+f)∗g

我们需要转换为

abc∗+de∗f+g∗+abc*+de*f+g*+ abc∗+de∗f+g∗+

中序转后缀表达式的策略就是：

  • 操作数：直接输出
  • 操作符：对于操作符我们使用一个栈来存储操作符。当遇到左括号时，直接入栈，遇到右括号时，弹出栈顶所有操作符并输出直到遇到左括号，弹出左括号，左括号不输出。当遇到其他操作符时，弹出栈顶操作符并输出直到栈顶操作符优先级低于改操作符，但是遇到左括号停止。其中±优先级较低，*/优先级较高

小行星碰撞

LeetCode 735

给定一个整数数组 asteroids，表示在同一行的小行星。

对于数组中的每一个元素，其绝对值表示小行星的大小，正负表示小行星的移动方向（正表示向右移动，负表示向左移动）。每一颗小行星以相同的速度移动。

找出碰撞后剩下的所有小行星。碰撞规则：两个行星相互碰撞，较小的行星会爆炸。如果两颗行星大小相同，则两颗行星都会爆炸。两颗移动方向相同的行星，永远不会发生碰撞。

输入：asteroids = [4,5,-6,4,8,-5]

输出：[-6,4,8]

这道题乍一看比较新颖，但其实观察碰撞规律之后就可以发现非常简单。

第一个行星4，不知道其他行星，先放到容器，看第二个行星5，跟第一个同向，所以同样也放到容器，第三个行星-6，方向与前面两个不同，所以先看上一个行星5，所以碰撞之后剩下-6，然后再看第一个行星4，也会碰撞，所以最后只剩下-6，依次类推。可以很明显发现符合栈的后入先出的特点。

要注意的其实就是几个条件的判断，理清逻辑就很简单了。

每日温度

LeetCode 739

输入一个数组，他的每个数字是某天的温度。请计算每天需要等几天才会出现更高的温度。

输入：[35,31,33,36,34]

输出：[3,1,1,0,0]

解释：第一天温度是35，需要等三天才会出现更高问题36，所以对应输出为3，第四天温度为36，后面没有更高的温度，所以对应输出为0

这道题我们可以使用单调栈存储每天的下标，从栈底到栈顶温度递减，当栈顶遇到更高温度时出栈，栈中的数据表示还没有遇到更高的温度

直方图最大矩形面积

给定 n 个非负整数，用来表示柱状图中各个柱子的高度。每个柱子彼此相邻，且宽度为 1 。

求在该柱状图中，能够勾勒出来的矩形的最大面积。

输入：heights = [2,1,5,6,2,3]

输出：10

解释：最大的矩形为图中红色区域，面积为10

这道题大体上还有三个思路：

  1. 暴力解法：遍历每个柱，计算以改高度能构成的最大矩形面积

  2. 分治法：我们可以将题目分解问三种情况，先找出柱状图中最矮的高度，然后问题就变成了最大矩形存在在最矮柱的左边，或者右边又或者是以最矮的高度，宽度为总宽度的矩形。递归下去即可。

  3. 单调栈：我们先模拟计算过程，例如第一个节点高度为2，我们先放到容器里面，下一个节点为1，小于2，那么以2为高度的最大矩形就可以确定了，计算，然后将2移除，将1放进去，再看下一个节点5，大于1,这个时候有1,5为高度的最大矩形还不能确定，所以继续向后，6大于5，继续，下一个为2，很明显以5和6为高度的最大矩形也能确定，移除容器。

    所以，关于这个问题，我们可以用单调栈存储下标节点，当遇到比自己的高度小的节点出栈，并计算自己能构成的最大矩形的面积，遇到更大的则入栈。
