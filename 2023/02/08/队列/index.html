队列

队列的基础知识

队列是一种非常常见的数据结构，他的核心特点就是FIFO，先进先出。

栈可以分很多的种类

  • 阻塞队列和非阻塞队列：阻塞队列就是提供阻塞功能，即当插入数据时，如果队列已经满了，那么将会等待直到队列释放出新的空间，而非阻塞队列则会直接插入失败。
  • 有界队列和无界队列：有界队列值得就是有固定大小的队列，而无界队列就是没有设置固定大小的队列。
  • 普通队列：就是常见的先入先出的最普通的队列。
  • 双端队列：队列的头部尾部都可以同时出队入队的队列。
  • 优先队列：优先队列并不是先进先出，而是按照优先级高的先出队，采用二叉堆实现。
  • 延迟队列：当入队的元素达到指定的延迟时间才可出队。

关于队列，我们主要知道几个操作函数

数组实现队列

队列的应用

滑动窗口

数组中某一长度的子数组可以看成数组的一个窗口，例如给定一个数组[1,2,3,4,5,6,7]，那么子数组[2,3,4]就是其中一个大小为3的窗口，如果改窗口向右滑动一个数字，那么窗口就包含数字[3,4,5]，可以发现，如果继续滑动，那么没向右滑动一个数字，最右边便会插入一个数字并把最左边的数字删除。即最先进入窗口的数字最先删除，符合先进先出的原则，因此数组的滑动窗口可以用队列表示。

滑动窗口的平均值

LeetCode 346

给定一个整数数据流和一个窗口大小，根据该滑动窗口的大小，计算滑动窗口里所有数字的平均值。

实现 MovingAverage 类：

  • MovingAverage(int size) 用窗口大小 size 初始化对象。
  • double next(int val) 成员函数 next 每次调用的时候都会往滑动窗口增加一个整数，请计算并返回数据流中最后 size 个值的移动平均值，即滑动窗口里所有数字的平均值。

输入：inputs = ["MovingAverage", "next", "next", "next", "next"]，inputs = [[3], [1], [10], [3], [5]]

输出：[null, 1.0, 5.5, 4.66667, 6.0]

这道题就是比较简单的滑动窗口使用队列的一个案例

最近请求次数

LeetCode 933

写一个 RecentCounter 类来计算特定时间范围内最近的请求。

请实现 RecentCounter 类：

  • RecentCounter() 初始化计数器，请求数为 0 。
  • int ping(int t) 在时间 t 添加一个新请求，其中 t 表示以毫秒为单位的某个时间，并返回过去 3000 毫秒内发生的所有请求数（包括新请求）。确切地说，返回在 [t-3000, t] 内发生的请求数。

保证 每次对 ping 的调用都使用比之前更大的 t 值。

这道题和上一题类似，都是时间窗口的简单应用。这道题就是当队列头部与要插入的数据相差3000时，弹出队列

二叉树的广度优先搜索

二叉树的广度优先搜索是从上到下按层遍历二叉树

以上图为例，我们首先遍历节点8，然后依次遍历节点6，节点10，节点5，节点7，节点9，节点11。

我们通常使用队列来实现二叉树的广度优先搜索。从根节点开始，先把根节点放入队列，然后每次从队列中取出一个节点，如果该节点有左右子节点，则分别把子节点放到队列尾部。重复直到队列为空。

假设该二叉树总共有n个节点，那么我们通过队列遍历所有节点的时间复杂度为\(O(N)\)，队列中最多存储的节点为倒数第二层遍历结束，且最后一层为满的情况，即最后一层节点最多为\((n+1)/2\)个，所以空间复杂度为\(O(N)\)

对于二叉树所有关于层方面的问题我们都可以使用队列实现广度优先搜索来处理。

在完全二叉树中添加节点

LeetCode 919

完全二叉树 是每一层（除最后一层外）都是完全填充（即，节点数达到最大）的，并且所有的节点都尽可能地集中在左侧。

设计一种算法，将一个新节点插入到一个完整的二叉树中，并在插入后保持其完整。

实现 CBTInserter 类:

  • CBTInserter(TreeNode root) 使用头节点为 root 的给定树初始化该数据结构；
  • CBTInserter.insert(int v) 向树中插入一个值为 Node.val == val的新节点 TreeNode。使树保持完全二叉树的状态，并返回插入节点 TreeNode 的父节点的值；
  • CBTInserter.get_root() 将返回树的头节点。

这道题就是经典的广度优先遍历的案例，因为是完全二叉树，所以我们一层一层搜索，直到发现一个节点不是既有左节点又有右节点，那么这个节点就是我们需要插入的节点的父节点。关于广度优先搜索还是使用队列即可。

二叉树中每层的最大值

LeetCode 515

给定一棵二叉树的根节点 root ，请找出该二叉树中每一层的最大值。

这道题又是涉及到每层的一个最大值，所以我们肯定还是使用广度优先遍历。其次就是如何判断每一层的开始和截止，这里我们有两种方式进行判断：

  1. 一个队列，使用current和next来记录当前行的节点数和下一行的节点数。

    root节点非空入队，current++，出队，current--，当存在左右节点时，左右节点入队，并且next加上入队的节点个数，当current为0的时候则表明当前层已经遍历结束，准备开始遍历下一层，那么current=next，next=0，如此重复直到队列为空

  2. 使用两个队列，queue1存放当前层的节点，queue2存放下一层的节点，当queue1为空时，令queue1=queue2，queue2为新的空队列

二叉树最底层最左边的值

LeetCode 513

给定一个二叉树的 根节点 root，请找出该二叉树的 最底层 最左边 节点的值。

假设二叉树中至少有一个节点。

输入：root = [2,1,3]

输出：1

题目中要求我们找到最底层的最左边的值，又是按层遍历，所以必然是广度优先搜索。我们仍然可以使用上一题使用的双队列，并且使用一个节点记录当前层最左边的节点。

二叉树的右侧视图

LeetCode 199

给定一个二叉树的 根节点 root，想象自己站在它的右侧，按照从顶部到底部的顺序，返回从右侧所能看到的节点值。

这道题看起来新颖，其实很简单，就是上一题的变种，即求二叉树每层最右边的值。我们还是可以使用两个队列对二叉树进行广度遍历，然后使用一个节点指向每层的最右边的节点。
